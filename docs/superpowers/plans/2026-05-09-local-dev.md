# Local Dev Environment Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Enable full end-to-end local testing of the TSG reseller portal without connecting to any external APIs (TeamLeader, Supabase Auth, Azure Blob).

**Architecture:** A `LOCAL_DEV=true` env flag activates in-process mocks throughout the FastAPI backend (auth bypass returns hardcoded dev reseller, TL client returns fake UUIDs, Blob client returns localhost URL). The Next.js frontend skips Supabase session fetching when `NEXT_PUBLIC_DEV_BYPASS=true`. PostgreSQL runs via Docker Compose and is seeded with realistic dummy data.

**Tech Stack:** FastAPI, SQLAlchemy async, PostgreSQL (Docker Compose), Next.js, Python asyncio, pytest

'use client'
import { createClient } from './supabase'

export class ApiError extends Error {
  constructor(public readonly status: number, message: string) {
    super(message)
    this.name = 'ApiError'
  }
}

export async function apiFetch(path: string, options?: RequestInit): Promise<Response> {
  const isLocalDev = process.env.NEXT_PUBLIC_DEV_BYPASS === 'true'

  let authHeader: Record<string, string> = {}
  if (!isLocalDev) {
    const supabase = createClient()
    const { data: { session } } = await supabase.auth.getSession()
    if (session) {
      authHeader = { Authorization: `Bearer ${session.access_token}` }
    }
  }

  const res = await fetch(process.env.NEXT_PUBLIC_API_URL + path, {
    ...options,
    headers: {
      'Content-Type': 'application/json',
      ...authHeader,
      ...options?.headers,
    },
  })

  if (!isLocalDev && res.status === 401) {
    const supabase = createClient()
    await supabase.auth.signOut()
    if (typeof window !== 'undefined') {
      window.location.href = '/reseller/login'
    }
    throw new ApiError(401, 'Session expired')
  }

  return res
}

export async function apiJson<T>(path: string, options?: RequestInit): Promise<T> {
  const res = await apiFetch(path, options)
  if (!res.ok) throw new ApiError(res.status, `API error ${res.status}`)
  return res.json() as Promise<T>
}
